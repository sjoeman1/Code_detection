
```python
from math import ceil

def get_digits_count(num):
    return len(str(num))

def get_max_integer(A, B, X):
    max_integer = 0
    for i in range(1, ceil(X / A) + 1):
        if A * i + B * get_digits_count(i) <= X:
            max_integer = i
    return max_integer

if __name__ == "__main__":
    A, B, X = map(int, input().split())
    print(get_max_integer(A, B, X))
```

This Python3 code calculates the maximum integer that Takahashi can buy given the input A, B, and X. It first calculates the number of digits in an integer using the `get_digits_count` function. Then, it iterates through integers from 1 to the ceiling of X divided by A (to avoid unnecessary iterations) and checks if the price is within the budget. If it is, it updates the `max_integer` variable with the current integer. Finally, it prints the `max_integer` value.
