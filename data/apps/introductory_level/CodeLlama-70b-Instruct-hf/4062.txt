```py
def solve(n, prefixes, suffixes):
    ans = ""
    for i in range(1, n):
        if prefixes[i] in suffixes:
            ans += "P"
        else:
            ans += "S"
    for i in range(n, 2 * n - 1):
        if suffixes[i] in prefixes:
            ans += "P"
        else:
            ans += "S"
    return ans

if __name__ == "__main__":
    n = int(input())
    prefixes = []
    suffixes = []
    for _ in range(n - 1):
        prefixes.append(input())
    for _ in range(n - 1):
        suffixes.append(input())
    print(solve(n, prefixes, suffixes))
```

In the above code, we read the input values for `n` and the prefixes and suffixes. We then call the `solve` function, which iterates over the prefixes and suffixes and determines whether each one is a prefix or a suffix. The function returns the resulting string, which we print as the output.
