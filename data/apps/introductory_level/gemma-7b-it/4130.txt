```python
n = int(input())
a = list(map(int, input().split()))

a.sort()

# Find the maximum number of boxers whose weights are different
max_team_size = 1
current_weight = a[0]
for i in range(1, n):
    if a[i] - current_weight > 0:
        max_team_size += 1
        current_weight = a[i]

print(max_team_size)
```
