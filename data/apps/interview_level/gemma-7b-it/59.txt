```python
n = int(input())
a = list(map(int, input().split()))
s = input().split()

a_dict = {i: s[i - 1] for i in range(1, n)}
sorted_a = sorted(a)

for i in range(n):
    if a_dict[i] == '1':
        a[i], a[i + 1] = a[i + 1], a[i]

print('YES' if a == sorted_a else 'NO')
```

**Explanation:**

1. Create a dictionary `a_dict` to store the information about whether swapping the i-th element with the (i + 1)-th element is forbidden.
2. Sort the array `a` in ascending order.
3. For each element in the array, if the swapping operation is permitted, swap the element with the element at the same position in the sorted array.
4. If the array is sorted in ascending order, print YES. Otherwise, print NO.
