```python
n, b = map(int, input().split())

# Calculate the factorial of n and find the number of trailing zeros
fact_n = 1
num_zeros = 0
for i in range(1, n + 1):
    fact_n *= i
    while fact_n % b == 0:
        num_zeros += 1
        fact_n //= b

# Print the number of trailing zeros
print(num_zeros)
```
