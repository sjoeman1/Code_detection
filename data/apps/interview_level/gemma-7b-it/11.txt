```python
n, a, b, p, q = map(int, input().split())

# Calculate the multiples of a and b
multiples_a = [i for i in range(1, int(n**0.5) + 1) if i % a == 0]
multiples_b = [i for i in range(1, int(n**0.5) + 1) if i % b == 0]

# Find the common multiples
common_multiples = set(multiples_a) & set(multiples_b)

# Count the number of painted tiles
num_painted = len(common_multiples)

# Calculate the total number of chocolates
s = p * num_painted + q * (n - num_painted)

# Print the answer
print(s)
```
