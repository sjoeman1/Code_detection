```python
n = int(input())
a = list(map(int, input().split()))

a.sort()

# Find the minimum cost and the value of t that minimizes the cost
min_cost = float('inf')
for t in range(1, int(a[0]) + 1):
    cost = 0
    for i in range(n):
        cost += abs(a[i] - t)
    if cost < min_cost:
        min_cost = cost
        t_opt = t

print(t_opt, min_cost)
```
