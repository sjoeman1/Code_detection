```python
def check\_sequence(a, b, c):
if b == a:
return "YES"
else:
	i = 1
	while True:
		if a + i * c == b:
			return "YES"
		elif a + i * c < b:
			i += 1
		else:
			return "NO"

# Test cases
print(check\_sequence(1, 7, 3)) # YES
print(check\_sequence(10, 10, 0)) # YES
print(check\_sequence(1, -4, 5)) # NO
print(check\_sequence(0, 60, 50)) # NO
```

The function `check_sequence` takes three integers `a`, `b`, and `c` as input and returns "YES" if `b` appears in the sequence `s` and "NO" otherwise. The sequence `s` is generated by starting with `a` and repeatedly adding `c` to get the next element. The function checks if `b` is equal to `a` and returns "YES" if it is. If not, it increments a variable `i` and checks if `b` is equal to `a + i * c`. If it is, the function returns "YES". If `b` is greater than `a + i * c`, the function increments `i` again and continues the loop. If `b` is less than `a + i * c`, the function breaks the loop and returns "NO".
