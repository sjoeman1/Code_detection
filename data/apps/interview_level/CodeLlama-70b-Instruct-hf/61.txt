

Here is the Python 3 code that solves the problem:

```python
# Input
n, b_x = map(int, input().split())
x = list(map(int, input().split()))
m, b_y = map(int, input().split())
y = list(map(int, input().split()))

# Convert x to decimal
dec_x = 0
for i in range(n):
    dec_x += x[i] * (b_x ** (n - i - 1))

# Convert y to decimal
dec_y = 0
for i in range(m):
    dec_y += y[i] * (b_y ** (m - i - 1))

# Compare x and y
if dec_x < dec_y:
    print('<')
elif dec_x > dec_y:
    print('>')
else:
    print('=')
```

The code first reads the input as specified in the problem statement. It converts the numbers `X` and `Y` from their respective bases to decimal and then compares them to determine the result.
