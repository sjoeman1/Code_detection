```python
m = int(input())

# Calculate the maximum number of blocks and the maximum X that results in that number of blocks
max_blocks = 0
max_x = 0

for i in range(1, int(m**0.5) + 1):
    blocks = i**3
    if blocks <= m and blocks > max_blocks:
        max_blocks = blocks
        max_x = i * blocks

print(max_blocks, max_x)
```
