```python
import numpy as np

n, l, v_1, v_2, k = map(int, input().split())

# Calculate the time taken by each pupil to reach the place of excursion
time_taken = l / v_1

# Calculate the time taken to fill the bus
time_taken_bus = l / v_2

# Calculate the time taken to get all pupils on the bus
time_taken_embarkation = np.ceil(n/k) * time_taken_bus

# Calculate the total time
time_taken_total = time_taken_embarkation + time_taken

# Print the time taken
print(time_taken_total)
```
